AWS cURL Command Generator

This extension generates AWS Curl commands or cURL with SigV4 from an API, which can be easily copied to the clipboard with a simple right-click.

AWSCurl/Curl With SigV4 BurpExtension Burp Extension to create AWS Curl commands or cURL with SigV4 from an API Simple right-click functionality to bring up the extension contenxt menu and select the extension.
